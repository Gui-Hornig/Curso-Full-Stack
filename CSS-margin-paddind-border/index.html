Margin-Padding e Border

Margin: ; Aplica um espaçamento entre os elementos

Padding: ; Aplica um espaçamento dentro do meu elemento

Para definir as medindas de cada lado, as tag funcionam como um relógio da esquerda para direita

Margin: 30px 15px 28px 16px
Padding: 10px 16px 28px 19px

Quando utilizar Padding e Margin?

Deve utilizar Padding quando quiser espaçar entre um conteúdo do elemento e as bordas dele

Deve utilizar Margin quando quiser espaçar 2 elementos
